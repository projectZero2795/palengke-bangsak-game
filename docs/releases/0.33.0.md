## Security boundary

This release is a Shared Mode baseline, not dedicated-server anti-cheat. The
player-hosted room creator remains trusted, exact collision paths are not
server-simulated, and the Palengke API cannot verify a Photon-signed match
receipt. See the Phase 33 guide and known-issues register before competitive or
tournament use.
