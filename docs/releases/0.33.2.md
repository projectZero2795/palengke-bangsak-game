# Bang-Sak 0.33.2 WebGL retry timing hotfix

Released: 2026-07-12

## Problem

Release `0.33.1` correctly detected a transient Photon game-server socket
failure and displayed retry `2/2`, but its `Task.Delay` wait could stall under
Unity WebGL before the second `NetworkRunner` started.

## Fix

- The retry delay now uses a Unity real-time coroutine, which continues on the
  WebGL player loop.
- The direct EU secure Photon route, two-attempt limit, detailed last failure,
  and manual create/join controls remain unchanged.
- Players continue to use only `https://games.palengke.es`; release-version
  query parameters are not part of normal play.
